# NEXUS Digital Universe

A next-generation futuristic digital universe built with HTML, CSS, and JavaScript.

## Features

- Cinematic entry animation
- Infinite animated space background
- 3D floating particles and nebula effects
- Interactive galaxy environment
- Mouse-reactive parallax and glow trail
- Floating holographic panels
- Neon cyberpunk and liquid UI effects
- Rotating planets, portals, and energy nodes
- Particle explosions
- Voice welcome using browser speech synthesis
- Generated ambient audio toggle using Web Audio
- Day/night universe mode
- Unlockable hidden achievements
- Interactive AI companion storytelling
- Responsive desktop, tablet, and mobile design

## How to Run

Open `index.html` in a modern browser such as Chrome or Edge.

No installation, API key, server, or external asset is required.

## Project Structure

- `index.html` - NEXUS interface, entry portal, nodes, panels, and overlays
- `styles.css` - Space environment, glass/liquid UI, 3D effects, animations, responsive design
- `script.js` - Interactions, voice, audio, achievements, parallax, and particles
- `README.md` - Project overview
